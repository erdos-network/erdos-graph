Expand description

Minimal logger interface used throughout the project.

Implementors should be cheap to clone and thread-safe if used in multithreaded contexts. For stubs, implement log and optionally override convenience methods. TODO: consider adding Send + Sync bounds once we finalize the threading model. The logger trait used across the project.

Implementations must be Send + Sync so they can be shared across threads when the global facade is used.
